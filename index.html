About the Mods

SMaTRS TOWERTIME! is a mod that focuses on change the game so it can be like Pizza Tower.

in the other side, Lumi's Mod only changes the levels and only has the Lapping System from TOWERTIME.

New stuff: Level edits, "Pizza Time" gimmick at the end of a level, Lap 2 and 3, Pizzafying some levels like 1-8, And much more!

About the Novemb3r Build

The Novemb3r (or November third) Build of TOWERTIME! Mod will be a build made to test the new features of the mod, it will NOT contain the tower, instead, it will have a custom hub level to play the levels made for the build.
